## ✨ Features

- **Clean home screen** — Clock, date, and weather at a glance with beautiful typography
- **Smart app access** — Pinned apps or most-used apps directly on home
- **Fast app drawer** — Fully alphabetical with real-time search
- **Productivity feed** — Upcoming calendar events, tasks, and screen time insights
- **Transparent background** — Full support for live wallpapers and transparency
- **Lightweight & fast** — Built for performance with minimal permissions
- **Bilingual ready** — English and Bahasa Indonesia

2. Install on your Android device (Android 6.0+)
3. Set Flow as your default launcher
